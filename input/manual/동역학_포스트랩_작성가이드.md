# 동역학 포스트랩 작성 가이드 (채점 기준 기반)

## 핵심 작성 원칙

아래 4가지 기준은 감점/가점의 주요 포인트다. 레포트 작성 시 각 항목을 체크리스트처럼 활용할 것.

---

### 1. 수식을 실제 실험 결과에 적용

단순히 수식을 소개하는 것으로 끝내지 않는다.
**수식이 이번 실험에서 어떻게 계산되었고, 결과에 어떻게 반영되는지** 구체적으로 서술해야 한다.

> **작성 방법**
> - 수식에 실험에서 측정한 실제 수치를 대입하여 계산 과정을 명시
> - "이 수식에 의하면 우리 실험에서는 __ = __ × __ = __ [단위]로 계산된다" 형태
> - 계산 결과가 실험 데이터(측정값, 그래프 등)와 어떻게 연결되는지 설명

---

### 2. 오차 분석 — 식 나열을 넘어 개선된 계산 결과 제시

오차 공식을 나열하거나 오차율만 계산하는 수준에서 멈추지 않는다.
**오차 원인을 분석한 뒤, 보정 또는 개선된 계산 결과까지 제시**해야 한다.

> **작성 방법**
> - 이론값 vs. 실험값 비교 → 오차율 계산 (기본)
> - 오차의 주요 원인 파악 (마찰, 공기저항, 측정 불확도 등)
> - 원인을 반영하여 수정된 이론값 또는 보정된 실험값을 재계산하고 제시
> - 보정 전/후 오차율을 나란히 비교

---

### 3. 수정된 수식 제안 및 실험 결과 보완

수식 도입 후, 실험 조건(비이상적 요인)을 반영한 **수치적으로 수정된 수식을 제안**하고,
이를 이용해 실험 결과를 보완·재해석한다.

> **작성 방법**
> - 이상적 가정(마찰 없음, 강체 등)이 실험에서 어떻게 위반되는지 명시
> - 보정 항(term)을 추가한 수정 수식 제시 (예: 마찰력 항 추가, 관성 보정 등)
> - 수정 수식으로 계산한 결과와 원래 결과를 비교하여 개선 여부 확인
> - 수식 제안은 LaTeX 형식으로 명확히 작성

---

### 4. 수식의 실험 적용 가능성 판단 및 적용 예시

이론에서 유도된 수식이 **이번 실험 조건에 실제로 적용 가능한지 먼저 판단**하고,
적용 가능한 경우 구체적인 적용 예시를 제시한다.

> **작성 방법**
> - 수식의 가정 조건(assumption) 명시
> - 실험 조건이 해당 가정을 만족하는지 검토 (정량적으로 판단)
> - 적용 가능하면: 수식에 실험값 대입 → 결과 해석
> - 적용 불가하거나 제한적이면: 어떤 조건에서 적용 가능한지, 어떻게 수정해야 하는지 서술

## 체크리스트 (레포트 제출 전 확인)

- [ ] 모든 수식에 실험 측정값이 대입되어 있는가?
- [ ] 오차율만 계산하고 끝내지 않았는가? (개선 결과까지 있는가?)
- [ ] 비이상적 요인을 반영한 수정 수식이 있는가?
- [ ] 수식의 가정 조건과 실험 조건을 비교·판단한 내용이 있는가?
- [ ] 고찰이 "실험과 무관한 일반론"이 아닌 "이번 실험 데이터에 근거한 구체적 분석"인가?

## 에이전트 활용 노트

이 파일은 동역학 포스트랩 레포트 작성 시 `orchestrator` → `report-writer` 에이전트가
고찰 및 결과 분석 섹션을 작성할 때 반드시 참조해야 한다.
특히 **고찰(Discussion)** 섹션과 **오차 분석** 섹션의 서술 깊이 기준으로 사용한다.
